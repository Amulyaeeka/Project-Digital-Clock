# Project-Digital-Clock
 To learn the basics of digital electronics, microcontrollers, and display interfacing by building a practical device that accurately shows time.

# Digital Clock in Verilog

## Modules
- Clock Divider
- Seconds Counter
- Minutes Counter
- Hours Counter
- Testbench

## Files
- Digitalclock_clkdiv.v
- Digitalclock_seconds.v
- Digitalclock_tb.v

## Features
- 24-hour format
- Reset support
- Clock division from FPGA clock to 1 Hz

## Simulation
Tested using Verilog testbench.
